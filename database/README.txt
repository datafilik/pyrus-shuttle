README: On database setup
**********************************************

Possible ways to regain the database schema includd

1. MANUALLY - create a database called 'doc_cabinet' and then use the SQL queries in the SQL files
	      in this directory to recreate the tables one after the other using phpMyAdmin (PMA). 
	      or any SQL software utility you see fit. PMA usually comes with a LAMP stack, e.g XAMPP

2. PMA IMPORT - Import the 'innodb_table_stats.ibd' file in this directory using PMA.
