**********************************************
README: On database setup
**********************************************

Possible ways to regain the database schema include

1. MANUALLY - involves creating a database called 'doc_cabinet' and then using the SQL queries in the SQL files
	      in this directory to recreate the tables using phpMyAdmin (PMA) which usually comes with a LAMP stack
	      e.g XAMPP

2. PMA IMPORT - Import the 'innodb_table_stats.ibd' file in this directory to PMA.
